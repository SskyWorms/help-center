How to Manage Tokens in Holdstation Wallet

Holdstation Wallet allows users to customize their token list by importing new tokens or hiding unwanted ones. This is managed via the gear icon in the token section.

Steps to Import a Token

1. Open the Holdstation Wallet app.

2. Navigate to the Token Section on the main page.

3. Tap the gear icon in the top-right corner of the token section.

4. Select "Import Token".

5. Enter the token's contract address, or search for it by name.

6. Confirm the token details and tap "Add" to import the token to your wallet.

Steps to Hide a Token

1. Open the Holdstation Wallet app.

2. In the Token Section, tap the gear icon in the top-right corner.

3. Locate the token you want to hide.

4. Toggle the visibility switch or tap "Hide Token" to remove it from your main list.

5. The token will still be linked to your wallet but will no longer appear on the main screen.
